Cache service

A Go microservice to cache JSON response from Third-party APIs.

We may at times prefer to cache JSON response we get from a third-party API. A few reasons we may want to do this are:

  • Avoid hitting API rate-limit
  • Slow response time from third-party API
  • Results is aggregation of multiple third-party API calls
  • Reduce total number of HTTP requests to fulfill a user query
